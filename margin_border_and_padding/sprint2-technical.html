Margin, Border and Padding

Margin

Margin is the property that sets the space taht is outside the comtent, it is manage trought diferents propertys and values. We can set a margin around the element by using the property:

  • margin

Or we can specify witch side of the element we want to place the margin by using the property:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

To make this propertys work we have to give them values, and this can be done in diferent ways.

If we want to give the same margin all around the content:

  • margin: 10px;

If we want to give a specific value to a specific property

  • margin-top: 10px;
  • margin-right: 10px;
  • margin-bottom: 10px
  • margin-left: 10px;

Or we can give value to the propertys just with one line of command (it's going to be interpreteted clockwise starting from the top):

  • margin: 10px, 15px, 10px, 15px;

Border

Border is the property that allows us to surround the content with a "box", and specify the style, width and color of that "box". We can set a border around the element by using the propertys:

  • border-width
  • border-color
  • border-style (required)

We can specify witch side of the element we want to place the border by using the property:

  • border-top
  • border-right
  • border-bottom
  • border-left

We can also give values to this property, by doing:

  • border: 10px;

Or specify diferent values to a specific border, by doing:

  • border-left: 10px solid red;

Padding

Padding is the property that sets the space that is inside of any defined borders, by using the property:

  • padding

Or we can specify witch side of the element we want to place the padding by using the property:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

To make this propertys work we have to give them values, and this can be done in diferent ways.

If we want to give the same padding all around the content:

  • padding: 10px;

If we want to give a specific value to a specific property

  • padding-top: 10px;
  • padding-right: 10px;
  • padding-bottom: 10px
  • padding-left: 10px;

Or we can give value to the propertys just with one line of command (it's going to be interpreteted clockwise starting from the top):

  • padding: 10px, 15px, 10px, 15px;
